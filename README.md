# 🛒 Customer Shopping Behavior Analysis

A complete Data Analytics project focused on analyzing customer purchasing behavior, shopping trends, and business insights using Python, SQL, and Data Visualization techniques.

---

# 📌 Project Overview

The objective of this project is to analyze customer shopping data and identify:

- Customer purchasing patterns
- Spending behavior
- Product performance
- Revenue trends
- Customer segmentation
- Sales insights

This project demonstrates the complete Data Analysis workflow including:

✔ Data Cleaning  
✔ Exploratory Data Analysis (EDA)  
✔ SQL Analysis  
✔ Data Visualization  
✔ Business Insights  

---

# 🚀 Features

- Customer Behavior Analysis
- Revenue Analysis
- Product Category Analysis
- Monthly Sales Trend
- Customer Segmentation
- SQL-Based Business Queries
- Data Visualization using Python
- Business Insights & Recommendations
